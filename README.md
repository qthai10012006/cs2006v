# Cinema Booking System

## Giới thiệu dự án

Cinema Booking System là hệ thống quản lý rạp chiếu phim được xây dựng bằng Java theo hướng lập trình hướng đối tượng (OOP).

Hệ thống hỗ trợ:

* Đăng nhập / đăng ký tài khoản
* Chọn phim
* Chọn suất chiếu
* Chọn ghế
* Thanh toán vé
* Áp dụng voucher giảm giá
* Xuất hóa đơn vé xem phim

---

## Công nghệ sử dụng

* Java JDK 17
* Java Swing
* OOP (Object-Oriented Programming)
* Serialization (ObjectOutputStream)
